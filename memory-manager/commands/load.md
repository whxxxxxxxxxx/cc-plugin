# 记忆加载

参数：$ARGUMENTS

## 操作步骤

### 情况一：未提供参数

加载最近 7 天的所有记忆文件：

1. 列出 `./.claude/sessions/` 中符合 `YYYY-MM-DD-*.md` 模式的所有文件
2. 筛选日期前缀在最近 7 天内的文件（包括今天）
3. 读取每个匹配的文件并显示其内容

### 情况二：提供了参数

参数为逗号分隔的文件名列表，例如 `2026-01-01-topic.md,2026-01-02-other.md`

1. 通过逗号分割并去除空格来解析 $ARGUMENTS
2. 对每个文件名，读取 `./.claude/sessions/<filename>`
3. 显示每个文件的内容

## 输出格式

对于每个加载的记忆文件，显示：

```
### <filename>
<文件内容>
---
```

加载所有文件后，总结："已加载 N 个记忆文件。" 并简要列出加载的主题。

如果文件不存在，警告用户："文件未找到：`.claude/sessions/<filename>`"
